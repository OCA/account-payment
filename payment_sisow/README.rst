======================
Sisow Payment Acquirer
======================

This module was written to support payments via the Dutch payment acquirer `Sisow <https://sisow.nl>`_ in Odoo.

Configuration
=============

To configure this module, you need to:

#. go to Accounting / Configuration / Payments / Payment Acquirers
#. find the record with `provider` ``Sisow``
#. fill in your `merchant ID` and your `merchant key` as provided by Sisow
#. after testing, set the field `Environment` to ``Production`` and click the top right button to make the payment acquirer visible on the website

Known issues / Roadmap
======================

* currently, this module only supports Sisow's iDeal implementation
